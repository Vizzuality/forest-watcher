Forest Watcher é um aplicativo móvel desenvolvido para disponibilizar o monitoramento de florestas on-line dinâmico e os sistemas de alerta do Global Forest Watch (GFW) off-line e para o campo. Forest Watcher permite aos usuários acessar facilmente os dados de mudanças florestais do GFW em qualquer dispositivo móvel, navegar para áreas de mudanças detectadas e coletar informaçõesindependente da conectividade.
Este aplicativo fornece uma solução acessível e fácil de usar para identificar rapidamente áreas para patrulhas e investigações de campo; coletar evidências de mudanças florestais em campo; usar os dados que são levantados para tomar decisões de gestão e conservação com base em evidências; e investigar produtos de detecção remota no campo.

Recursos:
* Defina áreas de interesse para monitorarmonitorar
* Armazene diversos dados de mudanças florestas com base em satélite, como alertas de desmatamento em tempo real no seu dispositivo móvel
* Sobreponha camadas contextuas, como Áreas Protegidas e Concessões de Exploração de Madeira, ou envie outros conjuntos de dados
* Investigue alertas no campo e colete informações (incluindo pontos de GPS e fotos) através de formulários embutidos e customizáveis
* Verifique, analise e baixe dados coletados através do app
* Gerencie uma equipe e mais com o aplicativo suplementar para computador através de forestwatcher.globalforestwatch.org.
